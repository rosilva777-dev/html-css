Usando links externos

Obs:Para colocar links externos no seu site, basta você selecionar um trecho do texto e envelopar buscando pela tag a href="digitando o link externo" no exemplo abaixo.

Lembrando:Para envelopar um trecho do texto, é preciso utilizar o comando Ctrl + Shift + P para abrir a paleta de comandos, digitar "Wrap with Abbreviation" e selecionar a tag HTML que deseja usar. Alternativamente, se você estiver lidando com múltiplas linhas, pode selecionar as linhas e usar o atalho Ctrl + Shift + L para selecionar todas as linhas e depois digitar a tag.
